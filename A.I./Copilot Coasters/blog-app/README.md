# Blackpool Rollercoaster Blog

A simple ReactJS blog-style application created with `create-react-app`. It lists the operating roller coasters at **Blackpool Pleasure Beach** using Bootstrap cards and the official [RCDB](https://rcdb.com/4795.htm) for reference information.

## Technologies

- React (Create React App)
- Bootstrap 5 for UI styling

## Features

- Coaster cards display name, design, and an image (using placeholders).
- Layout adapted from the Bootstrap blog example: https://getbootstrap.com/docs/5.0/examples/blog/

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd blog-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```

The app will open at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
